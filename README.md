# LUMIÈRE 경영 대시보드 (프로토타입)

뷰티 브랜드 대상 데이터 대시보드 서비스의 프론트엔드 프로토타입입니다. 현재는 DB 연동 없이 샘플 거래 데이터를 기반으로 동작합니다.

## 구성
- `index.html` / `style.css` / `app.js` — 정적 대시보드 (빌드 도구 없음)
- `metrics.js` — 거래원장/마케팅비용/트래픽 원천 데이터로부터 13개 단일지표 + 4개 복합지표를 계산
- `data.js` — 샘플 데이터 (거래원장 340건, 6개월)

## 지표
- 후행지표: 기여이익률, 재구매율, LTV, 반품률, AOV 등
- 선행지표: 신규고객수, CVR, 장바구니이탈률, ROAS, 리뷰평점, 광고비, 방문수
- 복합지표: LTV/CAC, 단위경제, 마케팅 효율 지수, 기여이익률

각 카드를 클릭하면 산출 방식, 6개월 추이, 채널/카테고리별 세부 데이터를 모달로 확인할 수 있습니다.

## 로컬 실행
정적 파일이므로 별도 서버 없이 `index.html`을 열어도 되고, 로컬 서버로도 실행 가능합니다.

```
python -m http.server 8123
```

## 배포
Cloudflare Pages — 빌드 명령 없음, 출력 디렉터리 `/` (루트).
